CF7 - Custom Error Messages to Contact Form 7 to Enjoy Quick Solution. 

== Description ==

In Now a days, WordPress website users getting many difficulties in set custom error messages with their site’s Contact Form 7 (CF7), which is one of the most used and reliable WordPress Plugin with widespread presence over the web. So many users converts error using jQuery or javascript customizations.

However, time to leave all your worries behind. As a recently launched ‘CF7 – Custom Error Messages’ will allow you, WordPress developers, to put a Custom Error Message in your contact form 7 so users will set Error Message with different fields easily and quickly.

Using this CF7 – Custom Error Message, you can easily set Error Message according to fields in your Contact Form 7. 

Digging up further to its advantage, after using this plugin doesn’t make use of any type of jquery customization. You just need to add Messages from admin according to your required fields in your CF7. In order to use this plugin, make sure your Contact Form 7 is activated.


= CF7 - Custom Error Message Features =

*	You can customized your Contact form 7 Error Messages.
*	Allows you to set Different Error Messasge according to your fields.
*	Plugin can work in multiple forms on different page or single page.
*	The Plugin can take values for now input fields including textbox, textarea and Email.
	

== Plugin Requirement ==

PHP version : 5.3 and latest
WordPress   : Wordpress 3.0 and latest

== Installation ==

1. There are two ways you can install this WordPress Plugin. Either Open https://wordpress.org/plugins/cf7-Custom-error-messages/, which is the official WordPress Plugins’ directory page, or upload all files manually to your site’s server. click on the download button the page.
2. Now login to your WordPress site and activate the plugin. Then, select ‘edit’ option in ‘Contact Forms’.
3. Click on the ‘Generate Tag’ and select ‘PayPal Submit Button’ from the dropdown menu.
4. Fill in all the required fields to set up a PayPal payment gateway.
